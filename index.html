Xibo Players

Package repository for Xibo digital signage players

DNF Repository (Fedora 43)

RPM packages with GPG-signed metadata. Actual packages are served from GitHub Releases.

Quick setup

# Add the repository
sudo dnf config-manager addrepo \
  --set=baseurl=https://dnf.xiboplayer.org/xibo-kiosk/rpm/fedora/43/\$basearch/ \
  --set=gpgcheck=0 \
  --id=xibo-kiosk

# Install packages
sudo dnf install xibo-kiosk

APT Repository (Ubuntu 24.04)

DEB packages with GPG-signed Release files.

Quick setup

# Add the repository
echo "deb [trusted=yes] https://dnf.xiboplayer.org/xibo-kiosk/deb/ubuntu/24.04 all/" \
  | sudo tee /etc/apt/sources.list.d/xibo-players.list

# Install packages
sudo apt update && sudo apt install xibo-kiosk

Kiosk session scripts for Ubuntu-based signage players.
