Xibo Players - Package Repository

RPM and DEB packages for Xibo digital signage players.

Quick Setup (Fedora)

sudo dnf config-manager addrepo \
  --set=baseurl=https://dnf.xiboplayer.org/xibo-kiosk/rpm/fedora/43/\$basearch/ \
  --set=gpgcheck=0 \
  --id=xibo-players

sudo dnf install xibo-kiosk xiboplayer-electron
