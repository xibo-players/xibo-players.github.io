🎬 Qualsevol tipus de contingut

Vídeo, imatges, àudio, PDF, canals RSS en directe, rellotges, pàgines web i streaming — reprodueix pràcticament qualsevol format multimèdia a les teves pantalles.

📅 Programació intel·ligent

Programa campanyes per hora del dia, prioritat o ubicació. Configura contingut d'interrupció, zones superposades i llistes de reproducció activades per meteorologia des del teu CMS.

🔧 Desplegament fàcil

Grava una imatge, engega i connecta al teu CMS. Les pantalles es registren soles i reben actualitzacions automàticament — no cal configuració manual.

🔍 Monitoritza-ho tot

Veu què s'està reproduint a cada pantalla amb proves de reproducció, captures remotes, progrés de descàrrega i registres detallats — tot visible des del teu CMS.

📺 Video walls

Sincronitza múltiples pantalles en un sol video wall. Reproducció coordinada, transicions de dissenys i programació a totes les pantalles.

💿 Grava una imatge preparada

Les imatges Xibo Kiosk inclouen tot: el sistema operatiu, inici de sessió automàtic, un assistent de configuració, tots els reproductors, monitorització de salut i actualitzacions automàtiques. Tries quin reproductor executar durant la configuració — i pots canviar en qualsevol moment. Tria el teu maquinari:

PC o portàtil

ISO instal·lador

Grava la ISO a un USB amb Balena Etcher o dd. Arrenca des d'USB — l'instal·lador formata el disc i ho configura tot automàticament. Treu l'USB i reinicia.

Després d'arrencar

Inicia sessió amb l'usuari xibo / contrasenya xibo. Canvia la contrasenya amb passwd. Connecta't al teu Xibo des de la pantalla de configuració del quiosc — la pantalla es registra automàticament i comença a reproduir el contingut programat.

📦 Instal·la en un Linux existent

Instal·la xiboplayer-kiosk al teu sistema Fedora o Ubuntu existent. Configura l'inici de sessió automàtic, una sessió de quiosc, monitorització de salut i el reproductor Electron — sense reinstal·lar el sistema operatiu.

# Afegeix el repositori
sudo dnf install \
  https://dl.xiboplayer.org/rpm/fedora/43/noarch/xiboplayer-release-43-2.noarch.rpm

# Instal·la el quiosc
sudo dnf install xiboplayer-kiosk

# Activa la sessió de quiosc
sudo systemctl set-default graphical.target
# Afegeix el repositori i la clau GPG
curl -fsSL https://dl.xiboplayer.org/deb/DEB-GPG-KEY-xiboplayer | \
  sudo gpg --dearmor -o /etc/apt/keyrings/xiboplayer.gpg
echo "deb https://dl.xiboplayer.org/deb/ubuntu/24.04 ./" | \
  sudo tee /etc/apt/sources.list.d/xiboplayer.list

# Instal·la el quiosc
sudo apt update && sudo apt install xiboplayer-kiosk

Què s'instal·la

El paquet xiboplayer-kiosk configura un usuari de quiosc dedicat amb inici de sessió automàtic, configura una sessió a pantalla completa i activa la monitorització de salut i les notificacions a pantalla. Instal·la un o més paquets de reproductors per separat — el quiosc utilitza el sistema d'alternatives perquè puguis canviar de reproductor en qualsevol moment. Reinicia per iniciar el quiosc.

Imatges de quiosc

Imatges d'arrencada predefinides — no cal instal·lar Linux. Descarrega, grava a USB o targeta SD, arrenca i el teu quiosc està llest. Consulta Primers passos per a instruccions pas a pas.

Tria el teu reproductor

Tria el reproductor que s'adapti al teu maquinari. Electron i Chromium utilitzen el mateix motor PWA — arexibo és un reproductor natiu independent.

⚡ Reproductor Electron

xiboplayer-electron

Aplicació d'escriptori autònoma — inclou el seu propi navegador, no cal instal·lar res més. La millor opció per a una configuració ràpida en qualsevol PC Linux. Es connecta al teu CMS automàticament.

RPM DEB x86_64 aarch64

🌐 Reproductor Chromium

xiboplayer-chromium

Reproductor lleuger per a sistemes que ja tenen Chromium o un navegador basat en Chromium instal·lat. Utilitza el navegador existent en lloc d'incloure'n un — ideal per a quioscs i dispositius encastats.

RPM DEB noarch

arexibo

arexibo

Reproductor natiu d'alt rendiment construït en Rust. Independent — no utilitza el motor PWA. Funciona tant en maquinari Intel/AMD com ARM.

RPM DEB x86_64 aarch64

Per a desenvolupadors i administradors de CMS

El motor i els components bàsics darrere de cada reproductor. La majoria d'usuaris no els necessiten — són per a integracions personalitzades i desplegaments autoallotjats.

📱 Motor PWA

@xiboplayer/pwa

El motor de renderització que impulsa tots els reproductors. També pot funcionar de manera autònoma a qualsevol navegador — desplega'l al teu propi servidor i apunta-hi un navegador. Gestiona vídeo, imatges, àudio, PDF, tickers, pàgines web, HLS i sincronització multipantalla.

npm Autoallotjat

🧱 @xiboplayer SDK

13 paquets npm

Paquets TypeScript modulars per a memòria cau, renderització, programació, sincronització multipantalla, comunicació amb el CMS (REST + SOAP), comandes en temps real, estadístiques i més. 1258 tests en 31 suites.

Configuració del repositori

Afegeix el repositori un cop i instal·la qualsevol paquet amb el gestor de paquets del teu sistema.

Un sol repositori serveix tots els paquets de xibo-players. Les metadades RPM s'allotgen aquí; els paquets reals es descarreguen directament des de les GitHub Releases de cada projecte.

1. Afegeix el repositori

# Instal·la el repositori i la clau GPG de xiboplayer
sudo dnf install \
  https://dl.xiboplayer.org/rpm/fedora/43/noarch/xiboplayer-release-43-2.noarch.rpm

2. Instal·la paquets

# Instal·la un o més paquets
sudo dnf install xiboplayer-kiosk
sudo dnf install xiboplayer-electron
sudo dnf install xiboplayer-chromium
sudo dnf install arexibo

3. Inicia la sessió de quiosc

# Activa l'inici de sessió automàtic a tty1 (reinicia per començar)
sudo systemctl set-default graphical.target
# La sessió de quiosc s'inicia automàticament al proper arrencada
Els paquets DEB es serveixen directament des d'aquest repositori. Actualment disponibles: xiboplayer-kiosk, xiboplayer-chromium, xiboplayer-electron, arexibo.

1. Afegeix el repositori

# Descarrega i instal·la el repositori i la clau GPG de xiboplayer
curl -fsSLO https://dl.xiboplayer.org/deb/ubuntu/24.04/all/xiboplayer-release_1.0.0_all.deb
sudo apt install ./xiboplayer-release_1.0.0_all.deb

2. Instal·la paquets

# Actualitza i instal·la un o més paquets
sudo apt update
sudo apt install xiboplayer-kiosk
sudo apt install xiboplayer-electron
sudo apt install xiboplayer-chromium
sudo apt install arexibo
